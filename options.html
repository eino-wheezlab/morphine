Charging

Set the optimal minutes balance charge rate for you! With the default settings, +1 minute will be given every 10 minutes, but it can be adjusted for optimal productivity, distraction, and attention span levels.

Targeting

Add domains to block and (optionally) allow, one per line. Enter the domain only, so no slashes and stuff. For example, reddit.com is accepted, but http://reddit.com/ isn't.

Subdomains are accepted, and if not provided will be wildcarded. The allowed domains override the blocked domains and can be used to allow specific subdomains.

Blocking

Morphine's default "blocked!" page is the lovely nope.avi. Not a fan? Set it to the popup page, which lets you instantly use minutes and load the original destination. Or, enter your very own custom URL (can be a local file), but beware of infinite loops!

Translating

Are you fluent in both English and any other language? You can help! The process is simple, whether you're translating a new language or fixing errors in an existing one. All it involves is downloading a list of phrases, then translating and sending it back to me.

However, parts of Morphine use some quirky phrasing... But don't worry! You can translate as much or as little as you like; the default English will be displayed where a translation is lacking.
